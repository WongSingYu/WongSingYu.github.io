Display Rules

  • Each card contains an image, a title, and a description.
  • The image size must be 300x200 pixels.
  • The title must be less than 50 characters.
  • The description must be less than 200 characters.
  • The cards must be arranged in order.
  • Each row can display a maximum of 3 cards.
